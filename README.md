# Personal GitHub Page

This repository contains a one-page professional website:

- `index.html`
- `styles.css`

## Publish to GitHub Pages

1. Create a new GitHub repository (for example: `ncesped.github.io`).
2. Push this folder to that repository.
3. On GitHub, open `Settings` -> `Pages`.
4. In **Build and deployment**, choose:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` and `/ (root)`
5. Save. Your site will be live in a few minutes.

If your repository is named `ncesped.github.io`, your URL will be:

`https://ncesped.github.io`

## Optional: Add downloadable CV PDF

If you want a download button, export your LaTeX CV as `cv.pdf` and place it in this repository root.
